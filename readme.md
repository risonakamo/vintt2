remake of VINTT program time-tracker.

# installing
```bash
npm i
npm run build
npm link
```

# usage
```
vintt2
```
begin watching programs. press enter, q, or c/ctrl+c to quit.

```
vintt2 add <exe name> <program name>
```
add the specified exe name to the watch list, with the given name

```
vintt2 show
```
show recorded times
